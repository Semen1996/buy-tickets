# Проект: Покупка билетов на теплоход
Сайт на GitHub Pages: https://semen1996.github.io/buy-tickets/ 

Страничка сайта, на которой пользователь выбрав направление, время и количество билетов может посчитать итоговые значения: общую стоимость, время в пути и т.д.

## Функционал:

* Выбор направление
* Выбор отправления. Если выбрано направление "из A в B и обратно в А", то показывается дополнительный список, в котором можно выбрать обратное время отправления из пункта B в пункт А, при этом время не пересекается (выдаются только те пункты, на которые юзер сможет успеть). 
* Ползователю прелагается выбрать количество билетов и тогда активируется кнопка "Посчитать".
* При клике на кнопку "Посчитать" показавается результат с направлением, временем в пути, временем отправления и временем прибытия.
